## How to build?

### Requirements

Tools:

* CMake: version 3.20
* MSVC: version i am lazy to check which minimal version of the compiler you need.

DLLs:

* `Xinput.dll`
* `DirectSound.dll`

### Steps

1. Run build script

```cmd
.\build.bat
```

2. Done
